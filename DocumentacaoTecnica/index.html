Configuração

Esta página ajudará você a instalar e criar seu primeiro aplicativo React Native usando o Expo Go.

Siga os seguintes passos:

  • Baixe o app Expo Go na sua loja de aplicativo
  • No seu notebook/PC, abra o prompt de comando
  • yarn create expo-app AwesomeProject
  • cd AwesomeProject
  • yarn expo start

Componentes

View é o componente mais fundamental para a construção de uma interface do usuário.

Text é o componente para exibir textos.

Image é o componente para exibir imagens.

TextInput é um componente fundamental para inserir texto no aplicativo por meio de um teclado.

StyleSheet é uma abstração semelhante ao CSS StyleSheets.

Interface

Esses controles de interface de usuário comuns serão renderizados em qualquer plataforma.

Button é um componente de botão básico que deve renderizar bem em qualquer plataforma. Oferece suporte a um nível mínimo de personalização.

Switch é um componente que renderiza uma entrada booleana.

Exibição de lista

Ao contrário do mais genérico ScrollView, os componentes de exibição de lista a seguir apenas renderizam os elementos que estão sendo exibidos na tela.

FlatList é um componente que age como uma interface de alto desempenho para renderizar listas simples e básicas

SelectionList é um componente que age como uma interface de alto desempenho para renderizar listas seccionadas
